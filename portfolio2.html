My Skills

HTML 5

HTML, or HyperText Markup Language is the standard markup language used to create web pages. It is a combination of Hypertext and Markup language. The Hypertext defines the link between web pages, and Markup defines the text document within tags to structure the web pages.

CSS 3

CSS stands for Cascading style sheets. It describes to the user how to display HTML elements on the screen in a proper format. CSS is the language that is used to style HTML documents. In simple words, cascading style sheets are a language used to simplify the process of making a webpage.

JavaScript

JavaScript is a programming language that adds interactivity to websites and web applications. It's used to create dynamic content, like interactive maps, animated graphics, and scrolling video jukeboxes. JavaScript is also used to control multimedia, respond to user events, and more.
